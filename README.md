# 🎁 Amigo Secreto - Java

## 📌 Sobre o Projeto

Um simples programa de **Amigo Secreto** feito em **Java**. Ele sorteia os participantes e informa quem é o amigo secreto de cada um.

## 🚀 Funcionalidades

- Cadastro de participantes
- Sorteio automático
- Exibição do resultado no console
